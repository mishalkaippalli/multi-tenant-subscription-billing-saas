# Multi-Tenant Subscription Billing SaaS

## Project Overview

A full-stack Multi-Tenant Subscription Billing SaaS application built using the MERN stack. The application allows multiple companies (tenants) to manage subscription plans, users, subscriptions, billing cycles, and dashboard analytics while ensuring complete tenant data isolation.

---

## Features

* Company Registration & Login
* JWT Authentication
* Role-Based Authorization (Admin/User)
* Multi-Tenant Data Isolation
* Subscription Plan Management
* User Management
* Subscription Assignment
* Billing Cycle Simulation
* Billing Logs
* Dashboard Analytics
